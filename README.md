BabyBloom 

BabyBloom is a full-stack eCommerce web application built using Spring Boot and JPA. 
It allows users to browse products, add items to cart or wishlist, and place orders. 
The application follows MVC architecture and uses Thymeleaf for dynamic frontend rendering.

Features
 
- User registration and login (Spring Security)
- Product listing and details page
- Add to cart functionality
- Wishlist management
- Order placement
- Admin product management

Tech Stack
  
- Java
- Spring Boot
- Spring Data JPA
- Spring Security
- Thymeleaf
- MySQL
- HTML, CSS, JavaScript,Bootstrap
  
Project Structure
  
- Controller
- Service
- Repository
- Entity
- Templates (Thymeleaf)
  
Future Enhancements
  
- Payment gateway integration
- Email notifications
- Product reviews and ratings
